== Description ==

A WooCommerce plugin that allow user to combine WooCommerce cart into the checkout page. 

In WooCommerce side menu, there will be a new submenu link called 'TT One-Page Checkout' where you can:

*   Enabled / Disabled the option
*   Position Cart Form at Checkout Page
*   Change "Add to cart" Button Text
*   Change "Order notes" Field Label
*   Change "Place order" Button Text
*   and many more customizations!


= Features =

*   Combine cart page into the checkout page
*   2 languages available : English US (en_US) and Chinese (zh_CN)

== Installation ==

1. Upload the entire *woocommerce-one-page-checkout* folder to the */wp-content/plugins/* directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'WooCommerce' at left sidebar menu and update the options at Tab *TT One-Page Checkout* there.
4. That's it. You're ready to go and cheers!
